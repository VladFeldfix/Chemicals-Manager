1. Download the application

PART III: MAIN MENU

1. PRINT YELLOW LABEL
 1.1. This will allow you to add a new LOT NUMBER to the database by filling a form
 1.2. At the end of the form press Y to submit the form and the new user will be added to the database
 1.3. When form is successfully submitted the application will open bar tender to print a yellow label
2. DELETE A USER
 2.1. This will allow you to delete an existing user
 2.2. Insert a valid user ID number to delete the user
3. EDIT A USER
 3.1. This will allow you to edit an existing user by filling a form
 3.2. At the end of the form press Y to submit the form and the user information will be edited
4. EXPORT
 4.1. This function allowes you to save your current database as a csv file
5. IMPORT
 5.1. This function allowes you to load users to your database from a csv file
 5.2. Warning! All current users will be deleted
6. LOAD FOLDER
 6.1. Loads a folder location
